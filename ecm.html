ECMとは

このページでは"ECM"とはどのようなものなのかについて基礎的な解説をしています。プログラムというよりかは理論寄りな部分になります。

ECMとは何？

ECMとは、Elliptic Curve Methodの略です。日本語では楕円曲線法といいます。

楕円曲線法とは、楕円曲線を用いて素因数分解するアルゴリズム(手法)のことです。


楕円曲線 \(y=x^3+ax+b\) のグラフ



私たちは、このECMを用いて素因数分解を行う活動しています。

それではECMではどのようにして楕円曲線を用いて素因数分解を行うのでしょうか。ECMでは【\(mod\) \(n\)での計算】と【スカラー倍算】についての2点が重要になります。

\(mod\) \(n\) の計算について

まず\(mod\) \(n\)とは簡単に「\(n\)で割った余り」とイメージしてください。

\(a\)と\(b\)の\(n\)で割ったときの余りが等しいとき、\(a≡b\) \((mod\) \(n)\)と表す。

例えば \(12+7≡4\) \((mod\) \(15)\)は「\(12+7(=19)\)を\(15\)で割った余りは\(4\)となる」と考えます。ECMでは素因数分解をする自然数を\(n\)と設定し、計算をすべて \(mod\) \(n\) 上で行います。

座標についても同様です。例えば\((19,31)\)はそれぞれの座標を\(15\)で割るとそれぞれ\(4\)と\(1\)となるので \(mod\) \(15\)上の世界では\((19,31)≡(4,1)\)となるので\((19,31)\)は\((4,1)\)として扱います。

ECMでは\(mod\)を取るようにしてプログラムを組むことが必須なのです。

スカラー倍算について

続いてスカラー倍算についてです。これはECMの基本的な処理の流れに大きく関係しています。

スカラー倍算ではまず以下の表現が重要になります。

  • \((1)\) 楕円曲線上のある点\(P\),点\(Q\)の通る直線の\(P\),\(Q\)以外の交点を\(P*Q\)と表す。
  • \((2)\) 点\(P*Q\)の\(x\)軸対称な点を\(P+Q\)と表す。(これを点\(P\)と点\(Q\)の加算といいます。)
楕円曲線上の点Pと点Qの加算

ECMではまず初めに楕円曲線上にある任意の1点を取って上の操作を行います。ある点\(P\)を通る接線の\(P\)以外の交点は上記の表現の\((1)\)において「点\(P\)と点\(P\)以外の交点」と置き換えて考えて\(P*P\)と表し、その点を\(x\)軸対称にした点を\(P+P(=2P)\)とします。(\(2P\)ができるので二倍算といいます。)

この後\(P\)と\(2P\)を加算した\(3P\)を求め、その後\(P\)と\(3P\)を加算した\(4P\)を求め…といった加算を繰り返すことができます。

ECMでは楕円曲線上のある点\(P\)を二倍算・加算によってある定数倍まで処理をする。
\(kP=P+P+P+…+P\) (\(k\) :自然数)

このように、点\(P\)と自然数\(k\)から\(kP\)を求めることをスカラー倍算といいます。

ECMではスカラー倍算が最も重要な演算処理です。 ( 理由は後述します。 )

まとめですが、ECMのスカラー倍の流れは以下のようになります。

  1. 初めに楕円曲線上に適当な点\(P\)を置く。
  2. 点\(P\)の接線を引く。
  3. 接線を引くと新たに楕円曲線との交点\(P*P\)ができる。
  4. 点\(P*P\) の\(x\)軸に対称な点となる\(2P\)をとる。(\(2P=P+P\))
  5. 生成された点\(2P\)と初めに置いた点\(P\)を直線で結ぶ。
  6. 直線を引くと新たに楕円曲線との交点\(2P*P\)ができる。
  7. 点\(2P*P\)の\(x\)軸に対称な点となる\(2P+P(=3P)\)をとる。

更に工程5~7を繰り返すことによって\(4P\)、\(5P\)、\(6P\)…と点を繰り返し生成する。


スカラー倍の流れ

ECMの流れ

実際に素因数分解をする際の流れは以下のようになります。

  1. 素因数分解したい整数を \(N\) とする。( 2.以降 \(mod\) \(N\)で演算処理をする )
  2. \(𝐸: 𝑦^2=𝑥^3+𝐴𝑥+𝐵\)（楕円曲線）と，それに乗る点\(P\)\((∈E)\)を決める。
  3. 適切な大きさの整数 \(𝐵1\) を与え，そこから \(𝑘\) を求める。
  4. \(mod\) \(N\) で，スカラー倍 \(𝑘𝑃\) を計算する。
  5. \(𝑘𝑃\) の座標と \(N\) の最大公約数を求める。最大公約数が \(1\) でなければ，\(N\) の因数の1つである。（運の要素が絡んでくる）
  6. 因数が見つからなければ， 2. に戻り，別の曲線と点で再度計算を行う。

スカラー倍算を行い、見つからなければ条件を変えて演算します。見つかるまでにはおおよそ数千回程度かかるので、ECMではスカラー倍算を効率化することが重要になります。2016年度では移動窓法と呼ばれる手法を用いることによって加算や二倍算の回数を減らしてスカラー倍算を行っています。

また、さらに効率化するためには射影座標等の理論的な知識が必要になります。

プロジェクトでは楕円曲線暗号入門を基礎的な楕円曲線の学習として活用していました。興味のある方は是非参照してみてください。
